Preguntes Freqüents (FAQ)

Quins coneixements previs necessite?

No es requereixen coneixements previs de programació. Este curs està dissenyat per a principiants.

Quant dura el curs?

El curs té una duració de 30 setmanes, amb una dedicació d'aproximadament 3 hores per setmana.

S'entrega certificat en finalitzar el curs?

No, este curs forma part d’un cicle formatiu, en el qual ja s’entrega un títol del cicle complet.

Quin programari necessite instal·lar?

Necessitaràs instal·lar Python i un editor de codi com Visual Studio Code o PyCharm. Les instruccions detallades es proporcionen en el primer mòdul.

Hi ha suport disponible durant el curs?

Sí, hi ha un fòrum de discussió i sessions de tutoria en directe per a resoldre dubtes i preguntes.
